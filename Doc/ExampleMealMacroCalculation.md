# Example Meal Macro Calculation (2P)

## Step 1: Get Each Component's 2P Portion Macros

### Basmati Rice with Corn
- Batch Totals: Calories: 4225, Fat: 50g, Protein: 79g, Carbs: 849g
- Cooked Weight: 2977g
- 2P Portion: 100g

**Per-gram values:**
- Calories: 4225 / 2977 ≈ 1.42
- Fat: 50 / 2977 ≈ 0.0168
- Protein: 79 / 2977 ≈ 0.0265
- Carbs: 849 / 2977 ≈ 0.2853

**2P (100g):**
- Calories: 100 × 1.42 = 142
- Fat: 100 × 0.0168 = 1.7g
- Protein: 100 × 0.0265 = 2.7g
- Carbs: 100 × 0.2853 = 28.5g
